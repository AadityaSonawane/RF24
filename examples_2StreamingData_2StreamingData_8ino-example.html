examples/StreamingData/StreamingData.ino

Written by 2bndy5 in 2020

A simple example of streaming data from 1 nRF24L01 transceiver to another.

This example was written to be used on 2 devices acting as "nodes". Use the Serial Monitor to change each node's behavior.

106

111  while (i < SIZE) {

113  if (!radio.writeFast(&buffer, SIZE)) {
114  failures++;

173 void makePayload(uint8_t i) {
174  // Make a single payload based on position in stream.
175  // This example employs function to save memory on certain boards.
176 
177  // let the first character be an identifying alphanumeric prefix
178  // this lets us see which payload didn't get received
179  buffer[0] = i + (i < 26 ? 65 : 71);
180  for (uint8_t j = 0; j < SIZE - 1; ++j) {
181  char chr = j >= (SIZE - 1) / 2 + abs((SIZE - 1) / 2 - i);
182  chr |= j < (SIZE - 1) / 2 - abs((SIZE - 1) / 2 - i);
183  buffer[j + 1] = chr + 48;
184  }
185 }
